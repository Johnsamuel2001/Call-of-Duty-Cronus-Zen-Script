⚙️ **How Almutawaa Works**

Almutawaa operates through a multi-layered controller input management system running directly on the Cronus Zen device.

The OLED menu is opened by pressing:

**L2 + Options**

Once inside the menu:

* D-Pad Up/Down navigates between mods
* D-Pad Left/Right changes values
* Cross enters advanced editing
* Circle exits and saves settings

The script processes recoil correction, aim assist behavior, movement automation, sensitivity adjustments, and utility functions simultaneously while adapting to the selected button and stick layouts.

Because all systems are modular, players can enable only the features they wish to use.

Every adjustment is saved automatically through Cronus Zen PVAR storage and remains active even after disconnecting the device.
